# Bluetooth HCI Captures

Place sanitized HCI snoop log captures here.

## Naming Convention

```
{bike_variant}_{firmware}_{feature}.log
```

Examples:
- `ntorq125xt_v2.3.1_connect_only.log`
- `ntorq125xt_v2.3.1_navigation.log`
- `ntorq_racexp_v3.0_call_alert.log`

## Sanitization Checklist

Before committing a capture, confirm:
- [ ] Bike MAC replaced with `AA:BB:CC:DD:EE:FF`
- [ ] No real phone numbers in notification payloads
- [ ] No personal names in notification payloads
- [ ] File is under 5MB (trim if needed)
